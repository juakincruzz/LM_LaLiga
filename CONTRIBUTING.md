# Guía de Contribución

Gracias por tu interés en contribuir a LM_LaLiga.

## Pasos para Contribuir

1. Fork el repositorio
2. Crea una rama para tu feature (`git checkout -b feature/descripcion`)
3. Haz commit de tus cambios (`git commit -m 'Descripción clara del cambio'`)
4. Push a la rama (`git push origin feature/descripcion`)
5. Abre un Pull Request

## Recomendaciones

- Mantén el código limpio y bien documentado
- Añade tests si es posible
- Actualiza el README si cambias funcionalidades
- Sigue el estilo de código existente
